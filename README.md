## 🧑‍💻 About Me

I am a Full Stack Engineer with a strong backend focus, working mainly with Go, Node.js, NestJS, Spring Boot, GCP, and Kubernetes. Most of my work is centered on microservices, APIs, event-driven flows, CI/CD pipelines, and operational improvements that keep systems easy to scale and easier to debug.

Across education, fintech, retail, healthcare, and telecom projects, I have worked on distributed architectures, contract and integration testing, cloud deployments, observability, and container hardening. Frontend is part of my toolkit, but backend engineering is where I generate the most leverage.

> *"I like systems that are simple to operate, hard to break, and clear enough that the next engineer can move fast."*

## 🔑 What I Bring

- 🏗️ **Backend architecture** — Go, NestJS, Spring Boot, gRPC, REST, GraphQL, and event-driven patterns.
- ☁️ **Cloud-native delivery** — Cloud Run, GKE, Pub/Sub, Cloud SQL, Cloud Storage, GitHub Actions, GitLab CI, and Cloud Build.
- ✅ **Quality-first workflows** — Karate DSL, k6, Dredd, Jest, Ginkgo, and pragmatic CI/CD guardrails.
- 📊 **Observability** — Tracing, metrics, logs, health checks, and production diagnostics using OpenTelemetry and cloud-native tooling.
